Auto Register

Classes that are required for all students or a specific group of students. Unlike Requirements, only one class can be listed per yaml entry.
Go to Schedule -> Automated to actually perform the registration.

Fields:
  • class: class name associated with class_id, for human use
  • class_id: the class id to automatically register for
  • applies_to:
  • - use empty list [] to specify all students
  • - current_grade to specify one grade level such as 8th, 7th, or 6th
  • - email for individual students
  • - group from Student Groups
  • exempt: list of emails who you don't want to auto register
  • id: autogenerated id for this auto requirement (not to be confused with class_id)
class_id is a required field.
