Auto Register

After entering the setup here, go to Schedule -> Automated and click "Add Auto Register Classes".
Example: 
- name: 6th Core
  class_id: 65
  applies_to:
    - current_grade: 6
  exempt:
    - leo.payne@gmail.com
- name: 7th Core
  class_id: 66
  applies_to:
    - current_grade: 7
    - email: leo.payne@gmail.com
    - group: Visiting_Group
Required fields:
  • class_id - must match id from Classes
  • id - generated, not the same as class_id
Optional fields:
  • class - reminder which class the class_id points to
  • applies_to
    • empty list [] for all students
    • current_grade specifies one grade level such as 8, 7, or 6
    • email for individual students
    • group from Student Groups
  • exempt - list of emails exempt from this class
