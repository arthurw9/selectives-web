Requirements

During registration, student schedules will be checked against requirements listed below. Additionally, the Student Schedule Errors report will generate a list of schedules that don't meet these requirements.
Classes listed under Auto Register will be checked separately and do not need to be relisted here.
No-repeat and pre-selected groups from Student Groups also do not need to be relisted here.

Fields:
  • name: label for this requirement
  • applies_to:
  • - use empty list [] to specify all students
  • - current_grade to specify one grade level such as 8th, 7th, or 6th
  • - email for individual students
  • - group from Student Groups
  • exempt: list of emails who are exempt from this requirement
  • class_or_group_options: the class or Class Groups that are required
Name is a required field.
