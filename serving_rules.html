Serving Rules

Grant student access to certain pages. Also controls the first page students see when they log in.
There are three types of pages:
  • materials - allows access the pre-registration materials
  • schedule - allows access to the schedule page.
  • final - allows access to the post-registration final schedule
If a student is granted access to multiple pages, the start page priority is: schedule, final, then materials.
In other words, if student has access to the schedule page, that will be their start page.
If schedule is not granted, then the first page they see will be final.
If final is also not granted, their start page will be materials.
Example: 
- name: materials
  allow:
    - current_grade: 6
    - current_grade: 7
    - current_grade: 8
- name: schedule
  allow:
    - current_grade: 7
    - current_homeroom: 19
    - email: sam.gamgee@gmail.com
- name: final
  allow:
    - current_grade: 7
    - current_grade: 8
Required field:
  • name
Optional field:
  • allow
    • current_grade specifies one grade level such as 8, 7, or 6
    • current_homeroom
    • email
