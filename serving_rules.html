Serving Rules

Grant student access. Also controls the start page - the first page students see when they log in.

There are three types of pages:
  • materials - allows access to the pre-registration materials
  • schedule - allows access to the schedule page.
  • final - allows access to the post-registration final schedule
If a student is granted access to multiple pages, the start page priority is: 1.) schedule, 2.) final, 3.) materials.
In other words, if a student has access to the schedule page, that will be their start page.
If schedule is not allowed, then their start page is final.
If final is also not allowed, then their start page is materials.
Example: 
- name: materials
  allow:
    - current_grade: 6
    - current_grade: 7
    - current_grade: 8
- name: schedule
  allow:
    - current_grade: 7
    - current_homeroom: 19
    - email: sam.gamgee@gmail.com
- name: final
  allow:
    - current_grade: 7
    - current_grade: 8
Required field:
  • name
Optional field:
  • allow
    • current_grade specifies one grade level such as 8, 7, or 6
    • current_homeroom
    • email
