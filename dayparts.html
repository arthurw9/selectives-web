Dayparts

Each daypart should be separate and not overlap with others.
Dayparts like M-W-F 8-9am are fine, but then don't also add Friday 8am-9am because it overlaps.
Example: 
- name: Mon A
  row: 1
  col: 1
  rowspan: 1
  colspan: 1
- name: Tues A
  row: 1
  col: 2
Required fields:
  • name - must be unique
  • row
  • col
Optional fields:
  • rowspan - defaults to 1, if not specified
  • colspan - defaults to 1, if not specified
